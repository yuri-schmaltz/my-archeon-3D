# Checklist de Acessibilidade (A11y) e UX — Hunyuan3D-2

## Acessibilidade
- Todos templates HTML possuem `aria-label` nos componentes principais
- Botões possuem labels descritivos
- Navegação por teclado funciona em todos elementos interativos
- Contraste mínimo garantido (WCAG AA)
- Imagens possuem atributo `alt` quando aplicável

## UX
- Estados de loading, empty, error, success e disabled implementados
- Mensagens de erro claras e acionáveis
- Feedback visual para ações críticas
- Responsividade mínima em templates
- Exemplos possuem instruções de troubleshooting

## Validação
- Testar navegação por teclado
- Usar ferramentas de validação WCAG (ex: axe, Lighthouse)
- Executar exemplos e verificar feedback visual
- Revisar templates em diferentes dispositivos

---
Atualizar checklist a cada release ou mudança relevante de UI/UX.
