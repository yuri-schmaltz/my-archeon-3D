# Archeon 3D

## Guia Oficial de Nome, Identidade e Domínio

Este documento define as diretrizes oficiais de nome, branding e
presença digital do projeto **Archeon 3D**, uma ferramenta de geração de
modelos 3D a partir de texto e imagens.

------------------------------------------------------------------------

## 1. Nome do Projeto

**Archeon 3D**

### Origem etimológica

-   Derivado do grego **archē (ἀρχή)**
-   Significado: origem, princípio, fundamento primordial
-   Representa:
    -   Geração
    -   Criação
    -   Base de tudo
    -   Conceito filosófico alinhado à ideia de modelagem a partir do
        "nada"

### Motivos da escolha

-   Sonoridade forte e profissional
-   Fácil pronúncia internacional
-   Conexão direta com criação e origem
-   Adequado para produtos de IA e computação gráfica

------------------------------------------------------------------------

## 2. Posicionamento da Marca

**Archeon 3D** é uma plataforma de IA focada em: - Geração procedural -
Conversão texto/imagem → 3D - Ferramentas para criadores, engenheiros e
artistas

### Valores

-   Inovação
-   Precisão técnica
-   Criatividade
-   Simplicidade
